# **App Name**: Futsal Future Survey

## Core Features:

- Animated Survey Form: Multi-step form with animated slides for collecting player feedback on their continuation for the next season.
- Player Name Selection: Dropdown menu on the second slide to select player name from a predefined list. Validation required to continue the form.
- Response Aggregation: Collection of responses within a JSON file to track players' decisions. Option to export.
- Interactive Team Roster: Landing page displays team roster. Clicking on a name shows whether they've completed the survey; if not, you may initiate it for them, which pre-populates their name on the second form page.
- Email Submission: Email integration to send collected data to guillaumepevrier@gmail.com upon survey completion. Uses emailjs.
- Real-time Result Display: Visual summary on a separate page to show which players have confirmed or declined using dynamic tables and color-coded indicators.
- AI Motivational Tool: AI-powered motivational message: Based on the player's response, the application displays a personalized, encouraging message. The application uses a tool that assesses the player's selection, injecting the appropriate type of motivation in the response.

## Style Guidelines:

- Primary color: Saturated purple (#A64AC9), embodying team spirit with its high energy
- Background color: Light purple (#EEE2FF), offers a gentle backdrop that enhances the dynamic text and graphical elements.
- Accent color: Blue-violet (#655DBB), this tone enhances both user experience and interface vibrancy while improving interaction.
- Headline font: 'Belleza' (sans-serif) for the tag/graffiti-style catchy texts.
- Body font: 'Alegreya' (serif) for form labels and instructions, ensuring clear readability.
- Use basketball-themed icons (ball, hoop, player silhouettes) to guide users.
- Fullscreen background videos on each slide related to Futsal highlights.
- Smooth transitions between form slides, mimicking the flow of a Futsal match.
